

* html5_surveillance

A browser-based video surveillance application built with Common Lisp metaprogramming. It captures camera feeds, processes them via WebGL shaders, streams via WebRTC peer-to-peer, and records processed video. The entire client-side application (JavaScript, GLSL shaders, HTML) is generated from Common Lisp at server startup  .

** Quick Start

1. Install SBCL and Quicklisp.
2. Load `cam.lisp` in your REPL.
3. The system auto-generates SSL certs, starts HTTPS on port 7777 and WebSocket on port 7778, and detects the server IP.
4. Open `https://<server-ip>:7777/`, accept the self-signed cert, then visit `wss://<server-ip>:7778/` to accept the WebSocket cert.
5. Grant camera permissions; video renders with WebGL processing; recording starts after 5 seconds; click Download to save the video .

** Architecture

- *Common Lisp generators* produce JavaScript, GLSL, and Python test scripts from S-expressions.
- *Server*: HTTPS (7777) + secure WebSocket (7778) with Clack and websocket-driver.
- *Client*: Camera access via getUserMedia, WebGL rendering, WebSocket signaling, WebRTC peer connections, MediaRecorder recording  .

** Core Capabilities

- Camera enumeration and stream access
- GPU-accelerated video processing (WebGL, 60fps)
- Real-time P2P streaming (WebRTC)
- WebSocket signaling for SDP exchange
- Video recording of processed canvas
- SSL/TLS (self-signed or Let's Encrypt)
- Automated Selenium-based tests  .

** Trident mode
- in conjunction with skewer-mode can execute parenscript
  in the browser. it does not support completion. it is on melpa
- if we don't use the parenscript package, macros are not expanded:
```
(@ (@
	((@ navigator media-devices get-user-media) constraints)
	(then got-stream))
       (catch handle-error))
```
- expands to this:
```
return at(at(at(navigator, mediaDevices, getUserMedia)(constraints), then(gotStream)), catch(handleError));
```
- using parenscript package gives:

```
    return navigator.mediaDevices.getUserMedia(constraints)[then(gotStream)][catch(handleError)];
```
